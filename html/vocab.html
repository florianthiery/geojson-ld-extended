GeoJSON+ Linked Data Vocabulary

See https://github.com/linkedgeodesy/geojson-plus-ld for vocabulary developments.

A RDF/XML description of this vocabulary is published at http://geojsonplus.linkedgeodesy.org/vocab.rdf.

This document is based on the GEOJSON-LD VOCABULARY by Sean Gillies (Mapbox) with a CC BY 4.0 license.

Core

GeoJSON+ Lied Data is based on the GEOJSON-LD VOCABULARY and uses all the GeoJSON Elements and properties.

Core Elements

none
